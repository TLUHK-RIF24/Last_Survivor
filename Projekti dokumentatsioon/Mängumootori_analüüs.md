## Uuritud tehnoloogiad

### 1. Unity

**Kirjeldus:**  
Unity on laialdaselt kasutatav mängumootor, mis toetab nii 2D kui 3D mänguarendust ning võimaldab eksportida mänge mitmele platvormile.

**Plussid:**
- Väga hea 2D tugi (sprite’id, animatsioonid, füüsika)
- Suur kogukond ja palju õppematerjale
- Asset Store valmis varade ja pluginate jaoks
- Lihtne buildida Windowsile, Macile ja WebGL-ile

**Miinused:**
- C#
- Võib olla algajale esialgu keerukas

---

### 2. Godot

**Kirjeldus:**  
Godot on avatud lähtekoodiga mängumootor, mis sobib hästi 2D mängude loomiseks.

**Plussid:**
- Tasuta ja avatud lähtekoodiga
- Lihtne ja kerge kasutada
- Hea 2D jõudlus

**Miinused:**
- Väiksem kogukond kui Unityl
- Vähem valmis varasid

---

### 3. HTML / CSS / JavaScript

**Kirjeldus:**  
Veebipõhine arendus HTML, CSS ja JavaScripti abil.

**Plussid:**
- Töötab otse brauseris
- Ei vaja eraldi mängumootorit
- Lihtne jagada

**Miinused:**
- Palju süsteeme tuleb ise nullist ehitada
- Keerulisem hallata suuremat projekti
- Piiratumad tööriistad võrreldes mängumootoritega

---

## Valitud tehnoloogia

Valisime oma projekti jaoks **Unity mängumootori**.

### Valiku põhjused

- Varasem kogemus Unity kasutamisega  
- Unity pakub tugevaid 2D tööriistu  
- Asset Store võimaldab kasutada valmis lahendusi  
- Võimalik eksportida nii arvutile kui ka WebGL versioonina  

Unity võimaldab kiiresti prototüüpida ning hiljem süsteeme laiendada.
